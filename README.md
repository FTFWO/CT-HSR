## Repository Setup

### Create a fresh conda environment, and install all dependencies.

```
conda create -n CT-HSR_env python=3.8
conda activate CT-HSR
git clone https://github.com/FTFWO/CT-HSR
cd HSRN
pip install -r requirements.txt
```
